# Cisco Packet Tracer Labs

A growing collection of Cisco Packet Tracer labs completed as part of my 
networking studies. Each lab folder contains the `.pkt` file, configuration 
screenshots and files.

## Labs

| # | Lab | Topics Covered |
|---|-----|----------------|
| 01 | [Cisco Routers and Switches Config](./lab-01-basic switching/) | IP Addressing, Router and Switches Configuration |
| 02 | [OSPF-Single Area Config] (./lab-02-OSPF-Singel Area/) | OSPF Configuration | 
| 03 | [OSPF-Multi Area Config] (./lab-03-OSPF-MultiAarea/) | IPV4 & IPV6 OSPF Configuration for Multi Area |
